# Code of Conduct

## The Short Version

Be kind. Be respectful. Help each other out.

VOQR is built on the belief that voice interaction with AI should be private, accessible, and available to everyone. We expect the same openness and respect in our community.

## Standards

**Do:**
- Be welcoming and inclusive
- Respect differing viewpoints
- Accept constructive feedback gracefully
- Focus on what's best for users

**Don't:**
- Harass, troll, or insult others
- Publish others' private information
- Use sexualized language or imagery
- Engage in conduct inappropriate for a professional setting

## Enforcement

Unacceptable behavior can be reported to the project maintainer via [InterGen Studios](https://intergenstudios.com). All reports will be reviewed promptly and confidentially.
